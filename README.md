# Лабораторная 2 по дисциплине "Технологии программирования"

## Постановка задачи

Разработать Django-приложение магазина товаров для быта с системой накопительных скидок для покупателей. Реализовать модульное тестирование и обеспечить работу приложения с базой данных PostgreSQL.

### Основные требования:
- Реализовать систему учета покупателей
- Внедрить накопительную систему скидок в зависимости от количества покупок
- Обеспечить модульное тестирование функциональности
- Настроить работу с PostgreSQL базой данных

## Краткое описание проекта

Django-приложение интернет-магазина товаров для быта с расширенной функциональностью системы лояльности. Покупатели получают накопительные скидки в зависимости от количества совершенных покупок.

### Основная функциональность:
- Каталог товаров для быта
- Система регистрации покупателей
- Накопительная система скидок
- История покупок
- Автоматический расчет скидок

## Используемые технологии

### Backend:
- **Python 3.14+** - основной язык программирования
- **Django 5.2** - веб-фреймворк
- **PostgreSQL** - система управления базами данных
- **psycopg2** - адаптер для работы с PostgreSQL

### Frontend:
- **HTML/CSS** - верстка интерфейса
- **Django Templates** - система шаблонов

### Дополнительные библиотеки:
- **dj-database-url** - работа с URL базы данных
- **gunicorn** - WSGI-сервер
- **whitenoise** - обслуживание статических файлов
- **pyyaml** - работа с YAML файлами

### Выводы по работе:
Проект демонстрирует разработку полнофункционального веб-приложения с системой лояльности, использованием современных технологий и тестирования.
